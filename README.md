# Firebot Combo Event

## IMPORTANT UPDATE - COMBOS REMOVED FROM TWITCH

Twitch has removed the "combo" feature effective March 31, 2026. [Link to Twitch announcement](https://help.twitch.tv/s/article/combos)

Therefore, this plugin is obsolete.

To remove this plugin from Firebot:

- Remove any events you may have created to handle combo events
- Go to Settings > Scripts > Manage Startup Scripts and delete the "Combo Event" script
- Restart Firebot
- Remove the `firebot-combo-<version>.js` script from your Firebot scripts directory
